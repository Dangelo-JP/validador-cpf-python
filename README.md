# Validador de CPF em Python

Aplicação desenvolvida em Python que verifica se um CPF é válido utilizando o algoritmo de cálculo dos dígitos verificadores.

O programa recebe um CPF digitado pelo usuário, remove caracteres especiais e realiza o cálculo matemático necessário para verificar sua validade.

---

## 🚀 Funcionalidades

- Recebe o CPF digitado pelo usuário
- Remove automaticamente pontos e traços
- Verifica se o CPF possui números sequenciais inválidos
- Calcula os dois dígitos verificadores
- Informa se o CPF é válido ou inválido
